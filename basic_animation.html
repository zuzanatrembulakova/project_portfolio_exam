Sprite Animation

This assignment was focusing on creating sprite animation in illustrator. I was inspired by a specific style of an artist and based on my inspiration I sketched my characters on paper and then I made them in illustrator. I made moodboard, so it would be easier for me to see my inspiration and design characters. I worked with different specific shapes and colors.

I design background, which consists of several layers and I put everything, character sprite, and background, together in CSS.

During this assignment, I gained better skills while working with illustrator and making an animation using @keyframes in CSS.

JavaScript controlled animation

The focus of the assignment was to create JavaScript controlled animation. I used my character from the previous task and applied ten different animations created in CSS on it.

It had to be obvious for the user how to turn the animation on and off and which animation was currently running. I included a sound when pressing a button.

I learned to implement sound in CSS and gained better skills in CSS animations and JavaScript.

While working on this assignment I was introduced to GitHub. I understood how to set up a project with Git and how GitHub works.

Project pool 1

This was a one-day assignment where we worked in pairs. We were given several projects to choose from. I and my peers decided to work on a Drum Machine where we created buttons that could be pressed using specific keys on one’s keyboard.

After pressing a certain key, the sound is played. It also had to be clear for the user which sound is playing – which button is active.

I gained better skills with JavaScript– I learned how to play the sound after pressing keys on the keyboard.

We were working in pairs and we had to be done with the project in one day, so it was important to decide and work fast and effectively.
