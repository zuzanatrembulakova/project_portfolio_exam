02.04.01 Responsive Website

The focus of this theme was to build a responsive website, i.e. a website that will be suitable for different screen sizes.

Each of us was given an art style (Art Deco in my case) which we had to include on our website. Firstly, I researched the given style, collected relevant information and images. I made moodboard based on this information and decided on typography and colors for my website.

I tested a prototype of the site created in Adobe XD using “trunk test“method.

While working on these assignments I gained useful skills on building a responsive website. I got better at coding and I was able to use more advanced CSS.

I learned a testing method – trunk test, which I was able to use in practice. I understood design principles, such as gestalt principles or colors schemes, and how important they are for clear and comfortable use of the website.
